# 📌 Transport Booking System - UI Flow

## 1. 🧭 Navigation (Sidebar)

🔗 https://www.figma.com/design/mzZyHsTBNyNnVZBe3UKTM0/Transport-Booking-System?node-id=96-489&t=wFCQltUnPSBP8DWg-4

### Luồng:

- Tại Home → click icon ☰ (góc trái)
  → mở Sidebar

- Từ Sidebar:
  - Trang chủ → quay về Home
  - Vé của tôi (My Booking) → chuyển sang My Booking
  - Các mục khác → điều hướng tương ứng
